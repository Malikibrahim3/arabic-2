## 🎮 Exercise Type Descriptions

### 1. **tap_letter** (23.4%)
**What it is:** Tap the correct letter/syllable/word from multiple choices

**Example:**
```
Prompt: "Tap the letter 'Baa'"
Choices: [ب] [ت] [ث] [ن] [ي] [ج]
```

**Used in:** All units, especially Units 1-2

---

### 2. **multiple_choice** (26.6%)
**What it is:** Select the correct answer from text choices

**Example:**
```
Prompt: "What is this letter called?"
Hint: ب
Choices: [Baa] [Taa] [Thaa] [Noon]
```

**Used in:** All units, most common exercise type

---

### 3. **hear_choose** (20.3%)
**What it is:** Listen to audio and select the correct option

**Example:**
```
Prompt: "Listen and select the correct letter"
Audio: [plays "ba"]
Choices: [بَ] [بِ] [بُ] [تَ]
```

**Used in:** All units, 20% of exercises (perfect!)

---

### 4. **trap_select** (5.0%)
**What it is:** Choose between highly confusable options

**Example:**
```
Prompt: "Careful! Which syllable makes 'ba'?"
Choices: [بَ] [بِ] [بُ]
Explanation: "Watch out for Kasra (ِ) which makes 'i'"
```

**Used in:** Units 1-4, concentrated in Unit 2 (17.7%)

---

### 5. **match_pairs** (8.8%)
**What it is:** Match items from two columns

**Example:**
```
Prompt: "Match each letter to its name"
Left: [ب] [ت] [ث]
Right: [Baa] [Taa] [Thaa]
```

**Used in:** All units, end of rounds

---

### 6. **word_assembly** (4.4%)
**What it is:** Tap letters in order to build a word

**Example:**
```
Prompt: "Build the word 'door'"
Choices: [ب] [ا] [ب] [م]
Correct order: ب → ا → ب
```

**Used in:** Units 2-6

---

### 7. **sentence_assembly** (3.4%)
**What it is:** Tap words in order to build a sentence

**Example:**
```
Prompt: "Build: 'This is a book'"
Choices: [هذا] [كتاب] [مدرسة]
Correct order: هذا → كتاب
```

**Used in:** Units 2-7

---

### 8. **introduction** (7.8%)
**What it is:** Intro cards explaining new content

**Example:**
```
Title: "This is ب (Baa)"
Content: "Name: Baa, Sound: /b/"
Action: [Continue]
```

**Used in:** All units, start of new content

---

### 9. **intro-trap-philosophy** (0.2%)
**What it is:** Special intro explaining trap exercises

**Example:**
```
Title: "The Trapdoor Challenge"
Content: "Watch out for confusable traps..."
Action: [Continue]
```

**Used in:** Unit 1 Round 3 (once per unit)
