## 🔍 Issues Identified

### ❌ Critical Issues

#### 1. Unit 4/5 Structure Confusion
**Problem:** Unit 4 and Unit 5 content are mixed together
- "Stage 4A Checkpoint" appears mid-Unit 4
- "Stage 5 Checkpoint" (unvowelled reading) appears in Unit 4 structure
- Unit 5 doesn't have its own clear unit test

**Impact:** 
- Students may progress to Unit 6 without mastering Unit 4 vocabulary
- Unvowelled reading (Unit 5) isn't properly gated

**Recommendation:** Restructure Units 4 and 5 clearly

---

#### 2. Missing Cumulative Tests
**Problem:** No tests that check retention across multiple units

**Current:** Each unit tests only its own content
- Unit 1 Test → Only letters
- Unit 2 Test → Only vowels
- Unit 3 Test → Only words from Unit 3

**Missing:** Tests that verify students remember:
- Letters (Unit 1) when learning words (Unit 3)
- Vowels (Unit 2) when reading sentences (Unit 6)
- Early vocabulary when learning conversations (Unit 7)

**Impact:** Students might forget earlier material as they progress

---

### ⚠️ Medium Priority Issues

#### 3. Inconsistent Naming
**Problem:** Tests are called different things:
- "Unit Test" (Unit 1, 2, 3)
- "Checkpoint" (Unit 4, 5, 6)
- "Final Test" (Unit 7, 8, 9)

**Recommendation:** Standardize naming for clarity

---

#### 4. Round 5 "Assessments" vs Unit Tests
**Problem:** Every node has a "Round 5: Assessment" but these aren't true gates

**Current Behavior:**
- Round 5 is just another lesson within a node
- Students can fail and still progress to next node
- Only Unit Tests actually gate progression

**Recommendation:** Either:
- Rename Round 5 to "Mastery Practice" (not "Assessment")
- OR make Round 5 a true gate that must be passed

## 🎯 Recommended Assessment Structure

### Ideal Structure for Each Unit:

```
UNIT X: [Topic]
├── Node 1: [Subtopic A]
│   ├── Round 1: Introduction
│   ├── Round 2: Practice
│   ├── Round 3: Discrimination
│   ├── Round 4: Review
│   └── Round 5: Mastery Practice (NOT a gate)
├── Node 2: [Subtopic B]
│   └── [Same 5 rounds]
├── Node 3: [Subtopic C]
│   └── [Same 5 rounds]
├── 📝 MID-UNIT CHECKPOINT (Optional, for long units)
│   └── Tests Nodes 1-3 content
├── Node 4: [Subtopic D]
│   └── [Same 5 rounds]
├── Node 5: [Subtopic E]
│   └── [Same 5 rounds]
└── 📝 UNIT TEST (Required, gates next unit)
    ├── Tests ALL unit content
    └── Includes refreshers from previous units
```

## 🎓 Pedagogical Rationale

### Why End-of-Unit Tests?
- ✅ Gate progression (prevent advancing without mastery)
- ✅ Consolidate learning (test all unit content together)
- ✅ Build confidence (students know they've mastered it)

### Why Cumulative Tests?
- ✅ Combat forgetting (spaced retrieval practice)
- ✅ Ensure retention (verify old content isn't lost)
- ✅ Build fluency (integrate old + new knowledge)

### Why NOT Make Round 5 a Gate?
- ✅ Too granular (would slow progress)
- ✅ Duolingo-style refreshers already handle review
- ✅ Unit tests are sufficient gates
